NoteScorer

Sheet Music library for musicians around the globe

This project will help musicians with varied interests to search, and add sheet music for their favourite piece of music. Users can create a secured account, like individual pieces, follow favourite musicians, showcase their ow pieces, and have that all stored in their account. The backend uses Musescore API to fetch data from a vast library. All new records will be stored in the NoteScorer database The project is a MEAN Stack application hosted on a Heroku server
